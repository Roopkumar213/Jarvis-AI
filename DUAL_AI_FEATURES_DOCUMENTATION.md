## Usage Tips

1. **Natural Speech**: You can use natural language - the system understands variations and synonyms.

2. **Specific Commands**: For file operations, be specific about locations (desktop, downloads, documents).

3. **Dynamic Values**: You can specify numbers for volume, brightness, and time-based commands.

4. **Chaining**: Some commands work better when Chrome or specific applications are already open.

5. **Error Handling**: If a command fails, try alternative phrasings or check if the target application is available.
